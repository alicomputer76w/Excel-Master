0.1 Introduction to MS Excel

Must Read

MS Excel is a spreadsheet program used for data storage, calculations, and analysis. (Excel ek spreadsheet program hai jo data save karne aur calculations ke liye use hota hai).

How to Open Excel?

  • Press Win + R, type excel and hit Enter.
  • Search 'Excel' in Start Menu.

The Excel Interface (User Interface)

Understanding the interface is the first step: (Excel interface ko samajhna pehla qadam hai):

  • Title Bar: Shows the workbook name.
  • Quick Access Toolbar: Common buttons like Save, Undo, Redo.
  • Ribbon: The top menu containing Tabs and Groups.
  • Tabs: Home, Insert, Page Layout, etc.
  • Groups: Commands related to each tab (e.g. Font, Alignment).
  • Formula Bar: Shows what is inside a cell.
  • Name Box: Shows the address of the selected cell (e.g. A1).

Basic Terminology

  • Workbook: An Excel file (collection of sheets).
  • Worksheet: A single page in a workbook.
  • Column: Vertical lines labeled A, B, C...
  • Row: Horizontal lines labeled 1, 2, 3...
  • Cell: Intersection of a Row and Column (e.g. B5).

1.1 IF & Nested IF (Logical Tests)

Essential

The IF function is the foundation of logical decision-making in Excel. It checks a condition and returns one value if TRUE, and another if FALSE. (IF function Excel mein faisla lene ke liye istemal hota hai. Agar condition sahi ho toh ek result, warna doosra).

Detailed Explanation: In advanced reporting, we often need multiple IFs nested together. This is called a "Nested IF". However, in modern Excel, you can also use the IFS function for cleaner code.

  • Syntax: =IF(logical_test, value_if_true, [value_if_false])
  • Scenario: You are calculating a sales commission. If sales > $5000, 10% commission; otherwise, 5%.
=IF(B2>5000, B2*10%, B2*5%)

Nested IF Example (Multiple Grades)

Multiple conditions: (Jab ek se zyada conditions hon)

=IF(B2>=90, "A+", IF(B2>=80, "A", IF(B2>=70, "B", "C")))

Practice Exercise

Write a formula to check student marks. If marks >= 33 return "Pass", else "Fail".

Solution: =IF(A2>=33, "Pass", "Fail")

1.2 VLOOKUP & XLOOKUP (Lookup Powerhouse)

VLOOKUP (Vertical Lookup) is used to find data in a large table based on a unique ID. XLOOKUP is the modern, faster, and more flexible version. (VLOOKUP kisi bade table se data dhoondne ke liye use hota hai. XLOOKUP iska naya aur behtar version hai).

Why XLOOKUP is better? Unlike VLOOKUP, XLOOKUP doesn't care if the column you're looking for is on the left or right. It also handles "Not Found" errors automatically.

=XLOOKUP(lookup_value, lookup_array, return_array, [if_not_found])
Common Mistake: In VLOOKUP, if you forget FALSE at the end, Excel might return the wrong data. Always use 0 or FALSE for exact matches.

1.3 INDEX + MATCH (The Professional Choice)

INDEX + MATCH is often preferred by advanced users because it's more stable and efficient than VLOOKUP. (Advanced users VLOOKUP ki jagah INDEX+MATCH use karte hain kyunke yeh zyada fast aur flexible hai).

How it works: MATCH finds the position (row number), and INDEX pulls the data from that position.

=INDEX(SalaryRange, MATCH(EmpID, IDRange, 0))

1.6 SUMIF & SUMIFS (Conditional Sums)

SUMIF adds up cells that meet one condition. SUMIFS is more powerful and can handle multiple conditions. (SUMIF ek condition par sum karta hai, jabke SUMIFS multiple conditions par).

Scenario: You want to find the total sales for the "East" region. Use SUMIF. If you want total sales for "East" region for "John" only, use SUMIFS.

=SUMIFS(sum_range, criteria_range1, criteria1, [criteria_range2, criteria2], ...)

Similar to SUMIF/SUMIFS, but they count the number of cells that meet conditions instead of summing them. (Yeh formulas cells ko ginte hain, sum nahi karte).

=COUNTIFS(criteria_range1, criteria1, [criteria_range2, criteria2], ...)

2.1 Advanced Sorting & Filter

Sorting helps you organize your data, but Advanced Filter allows you to perform complex extracts. (Sorting data ko tarteeb dene ke liye hai, lekin Advanced Filter se aap mushkil conditions par data extract kar sakte hain).

Advanced Filter Tip: You can use Advanced Filter to extract "Unique Records Only" to a new location. This is great for creating unique lists of customers or products.

Steps to use Advanced Filter:
  1. Select your data range.
  2. Go to Data tab.
  3. Click Advanced in Sort & Filter group.
  4. Specify the Criteria Range to filter specific rows.

2.2 Data Validation & Dynamic Dropdowns

Data Validation ensures that users only enter correct data. Dynamic dropdowns update themselves when you add new items to your list. (Data Validation se user ghalat entry nahi kar sakta. Dynamic dropdowns khud-ba-khud update hote hain).

How to make it Dynamic? Use Excel Tables (Ctrl+T) for your source list. When you add a row to the table, the dropdown will show it automatically.

How to create:

Data Tab > Data Validation > Allow: List > Source: (Select your items range)

Pivot tables are the most powerful data analysis tool in Excel. They let you summarize thousands of rows into a simple report. (Hazaron rows ka data chand seconds mein summarize karein).

Grouping Data: You can group dates into Months, Quarters, or Years. This is essential for time-based reports.

Calculated Fields: Add custom formulas inside your Pivot Table, like Profit = Sales - Cost.

Grouping Data:

Right-click any date field in Pivot Table > Group > Select Months/Years.

Calculated Field: =Sales * 0.10

6.1 Power Query (The Data Cleaning King)

Power Query is the most important automation tool in modern Excel. It records your data cleaning steps so you can refresh them with one click. (Data cleaning aur import karne ka sab se behtareen tariqa).

Unpivot Columns: If you have data spread across columns (Jan, Feb, Mar...) and want it in one column, use "Unpivot". This is a life-saver for data analysts.

How to use Power Query:
  1. Data Tab > Get Data > From File > From Workbook.
  2. Click Transform Data to open Power Query Editor.
  3. Apply steps like Remove Columns, Filter, Split.
  4. Click Close & Load.
