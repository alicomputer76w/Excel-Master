# Excel Mastery Hub

This is a comprehensive, single-page web application designed to teach advanced Excel skills. The platform is built with HTML, CSS, and JavaScript, and it is fully responsive for both desktop and mobile devices.

## Features

- **User Authentication:** Secure user registration and login system with WhatsApp OTP verification.
- **Interactive Course Modules:** A complete curriculum covering everything from basic formulas to advanced data automation.
- **Password Management:** Users can change their password and reset it via OTP if forgotten.
- **Progress Tracking:** The application tracks which modules a user has visited and displays a completion percentage.
- **Online Assessments:** Three levels of quizzes (Beginner, Intermediate, Expert) to test user knowledge.
- **Certificate Generation:** Users who pass all assessments can generate a professional certificate of completion.
- **Admin Dashboard:** A special section for administrators to view all registered users and certificate records.

## How to Run

1.  Clone the repository.
2.  Open the `index.html` file in your web browser.
3.  No special servers or dependencies are required. The entire application runs on the client-side using Local Storage to persist data.

## SEO & Deployment

- A `sitemap.xml` file is included to help search engines like Google index the site.
- To deploy, simply upload the `index.html`, `style.css`, and `script.js` files to any static web host.
